# Go::Zxing::Qrcode

Go QR Code encoder Ruby wrapper.

## Installation

Add this line to your application's Gemfile:

    gem 'go-zxing-qrcode'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install go-zxing-qrcode

## Usage

GoZXingQrcode::Encoder.generateQRCodeImage("string to encode", "image.png", 3,5)
